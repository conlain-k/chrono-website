chrono::vehicle::ChDataDriver Class Reference
VEHICLE module » Driver models

Description

Driver inputs from data file.

A driver model based on user inputs provided as time series. If provided as a text file, each line in the file must contain 4 values: time steering throttle braking It is assumed that the time values are unique. If the time values are not sorted, this must be specified at construction. Driver inputs at intermediate times are obtained through linear interpolation.

#include <ChDataDriver.h>
