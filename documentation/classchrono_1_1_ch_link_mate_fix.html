void chrono::ChLinkMateFix::Initialize ( std::shared_ptr< ChBodyFrame >  mbody1,
std::shared_ptr< ChBodyFrame >  mbody2 
)

Specialized initialization for "fix" mate, given the two bodies to be connected; the positions of the two auxiliary frames where the two bodies are connected are both automatically initialized as the current absolute position of mbody1.

Use ChLinkMateGeneric::Initialize() if you want to set the two frames directly.
