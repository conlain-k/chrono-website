chrono::fea::ChLinkPointPoint Class Reference
FEA module » Constraints

Description

Class for creating a constraint between two xyz FEA nodes (points).

That is, the two nodes will be joined, as overlapping. Nodes are 3-DOF points that are used in point-based primitives, such as ChMatterSPH or finite elements.

#include <ChLinkPointPoint.h>

Fetches the reactions from the lagrangian multiplier (l_i) of encapsulated ChConstraints.

Mostly used after the solver provided the solution in ChConstraints. Also, should convert the reactions obtained from dynamical simulation, from link space to intuitive react_force and react_torque.
