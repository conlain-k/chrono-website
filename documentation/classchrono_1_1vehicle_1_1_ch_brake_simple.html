chrono::vehicle::ChBrakeSimple Class Referenceabstract
VEHICLE module » Wheeled vehicles » Brake subsystem

Description

Template for a simple brake subsystem.

Simple brake created with constant torque opposing wheel rotation. It just uses a speed-dependant torque, so it fits in ODEs because it does not use DVI set valued constraints (the drawback is that it cannot simulate sticking brakes).

#include <ChBrakeSimple.h>
