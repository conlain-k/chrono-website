virtual void chrono::fea::ChElementBase::ComputeKRMmatricesGlobal ( ChMatrix<> &  H,
double  Kfactor,
double  Rfactor = 0,
double  Mfactor = 0 
)
pure virtual

Sets H as the stiffness matrix K, scaled by Kfactor.

Optionally, also superimposes global damping matrix R, scaled by Rfactor, and mass matrix M, scaled by Mfactor. Matrices are expressed in global reference. Corotational elements can take the local Kl & Rl matrices and rotate them. CHLDREN CLASSES MUST IMPLEMENT THIS!!!

Implemented in chrono::fea::ChElementHexa_20, chrono::fea::ChElementTetra_4_P, chrono::fea::ChElementTetra_10, chrono::fea::ChElementBeamANCF, chrono::fea::ChElementBeamEuler, chrono::fea::ChElementShellReissner4, chrono::fea::ChElementHexa_8, chrono::fea::ChElementShellANCF, chrono::fea::ChElementTetra_4, chrono::fea::ChElementBar, and chrono::fea::ChElementSpring.

virtual void chrono::fea::ChElementBase::GetStateBlock ( ChMatrixDynamic<> &  mD)
pure virtual

Fills the D vector (column matrix) with the current field values at the nodes of the element, with proper ordering.

If the D vector has not the size of this->GetNdofs(), it will be resized. For corotational elements, field is assumed in local reference! CHLDREN CLASSES MUST IMPLEMENT THIS!!!

Implemented in chrono::fea::ChElementTetra_4_P, chrono::fea::ChElementShellReissner4, chrono::fea::ChElementShellANCF, chrono::fea::ChElementBeamEuler, chrono::fea::ChElementHexa_20, chrono::fea::ChElementBeamANCF, chrono::fea::ChElementTetra_10, chrono::fea::ChElementHexa_8, chrono::fea::ChElementTetra_4, chrono::fea::ChElementBar, and chrono::fea::ChElementSpring.
