chrono::fea::ChBeamSectionAdvanced Class Reference

Description

Geometry for a beam section in 3D, along with basic material properties.

It also supports the advanced case of Iyy and Izz axes rotated respect reference, centroid with offset from reference, and shear center with offset from reference. This material can be shared between multiple beams.

#include <ChBeamSection.h>

Set the displacement of the shear center S respect to the reference beam line.

For shapes like rectangles, rotated rectangles, etc., it corresponds to the centroid C, but for "L" shaped or "U" shaped beams this is not always true, and the shear center accounts for torsion effects when a shear force is applied.
