chrono::fea::ChElementTetra_10 Class Reference
FEA module » Elements

Description

Tetahedron FEA element with 10 nodes.

This is a quadratic element for displacementes; stress and strain are interpolated depending on Gauss points.

#include <ChElementTetra_10.h>

values of shape functions at zi parametric coordinates, where r=1 at 2nd vertex, s=1 at 3rd, t=1 at 4th. All ranging in [0...1]. The last, u (=1 at 1st vertex) is computed form the first 3. It stores the Ni(r,s,t) values in a 1 row, 10 columns matrix.
