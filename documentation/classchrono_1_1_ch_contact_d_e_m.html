chrono::ChContactDEM< Ta, Tb > Class Template Reference

Description

template<class Ta, class Tb>
class chrono::ChContactDEM< Ta, Tb >

Class for penalty-based contact between two generic contactable objects.

Ta and Tb are of ChContactable sub classes.

#include <ChContactDEM.h>
