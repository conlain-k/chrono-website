chrono::ChFile_ps Class Reference

Description

Class for postScript(TM) output.

Defines special file class ChFile_ps for EPS or PS output, i.e. Encapsulated PostScript or PostScript (TM) files incuding vectorial plots, graphs, lines, fonts, etc.

#include <ChFilePS.h>
