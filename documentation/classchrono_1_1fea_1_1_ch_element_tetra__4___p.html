chrono::fea::ChElementTetra_4_P Class Reference
FEA module » Elements

Description

Tetahedron FEM element with 4 nodes for scalar fields (for Poisson-like problems).

This is a classical element with linear displacement. ***EXPERIMENTAL***

#include <ChElementTetra_4.h>

Fills the N shape function matrix with the.

values of shape functions at zi parametric coordinates, where z0=1 at 1st vertex, z1=1 at second, z2 = 1 at third (volumetric shape functions). The 4th is computed form the first 3. All ranging in [0...1]. NOTE! actually N should be a 3row, 12 column sparse matrix, as N = [n1*eye(3) n2*eye(3) n3*eye(3) n4*eye(3)]; , but to avoid wasting zero and repeated elements, here it stores only the n1 n2 n3 n4 values in a 1 row, 4 columns matrix!
