chrono::fea::ChLoaderBeamWrench Class Reference

Description

Atomic wrench.

Loader for a wrench (force+torque) at a specific position of a beam. An atomic load on the beam is a wrench, i.e. force+load aplied at a certain abscyssa U, that is a six-dimensional load. It is not a distributed load, so inherit it from ChLoaderUatomic.

#include <ChLoadsBeam.h>
