chrono::fea::ChElementGeneric Class Reference
FEA module » Elements

Description

Class for all elements whose stiffness matrix can be seen as a NxN block-matrix to be splitted between N nodes.

Helps reducing the complexity of inherited FEA elements because it implements some bookkeeping for the interface with solver. This means that most FEA elements inherited from ChElementGeneric need to implement at most the following two fundamental methods: ComputeKRMmatricesGlobal(), ComputeInternalForces()

#include <ChElementGeneric.h>
