chrono::ChClassRegister< t > Class Template Reference

Description

template<class t>
class chrono::ChClassRegister< t >

ChClassRegister<t> is responsable for registration of C++ classes at compile time (each class name ID enters a static list of items when the program starts).

During run time, it can be used in whatever moment to create classes given their name ID.

#include <ChClassRegister.h>
