Evaluates a tangent versor on a geometry, given parametric coordinates, if possible.

Parameters U,V,W should be usually in 0..1 range. For a line, only U parameter is needed, for a surface also V. Computed value goes into the 'pos' reference. It could be overriden by inherited classes if a precise solution is known (otherwise it defaults to numerical BDF using the Evaluate() function.
