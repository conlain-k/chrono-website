chrono::ChLoadableU Class Referenceabstract

Description

Interface for objects that can be subject to line loads, distributed along U coordinate of the object.

For instance finite elements like beams.

#include <ChLoadable.h>

Evaluate N'*F , where N is some type of shape function evaluated at U coordinate of the line, ranging in -1..+1 F is a load, N'*F is the resulting generalized load Returns also det[J] with J=[dx/du,..], that might be useful in gauss quadrature.
