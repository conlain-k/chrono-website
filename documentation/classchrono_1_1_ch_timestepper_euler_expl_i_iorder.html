chrono::ChTimestepperEulerExplIIorder Class Reference
Chrono::Engine » Time integrators

Description

Euler explicit timestepper customized for II order.

(It gives the same results of ChTimestepperEulerExpl, but performs a bit faster because it can exploit the special structure of ChIntegrableIIorder) This integrator implements the typical Euler scheme: x_new = x + v * dt v_new = v + a * dt

#include <ChTimestepper.h>
