chrono::ChAsset Class Reference
Chrono::Engine » Asset objects

Description

Classes for adding user data (such as rendering shapes, reference to files) to ChPhysicsItem objects.

User can inherit his classes for custom assets from this class.

#include <ChAsset.h>
