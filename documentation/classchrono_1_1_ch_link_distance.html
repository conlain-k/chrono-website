chrono::ChLinkDistance Class Reference

Description

Class for enforcing a fixed polar distance between two points on two ChBodyFrame objects.

The two points which are used to define the end points of the distance are assumed not to move respect to the two owner ChBody, as well as the amount of the distance is assumed not to change during the simulation. If you need to have a time-varying distance, or distance between two points which move respect to the bodies, please use the more advanced ChLinkLinActuator.
