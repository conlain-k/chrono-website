chrono::fea::ChVisualizationFEAmesh Class Reference

Description

Class for a FEA mesh visualization.

It converts tetahedrons, etc. into a coloured triangle mesh asset of class ChTriangleMeshShape that is contained in its sublevel, so that it can be rendered or postprocessed.

#include <ChVisualizationFEAmesh.h>
