chrono::ChGlyphs Class Reference

Description

Class for referencing a set of 'glyps', that are simple symbols such as arrows or points to be drawn for showing vector directions etc.

Remember that depending on the type of visualization system (POVray, Irrlicht,etc.) this asset might not be supported.

#include <ChGlyphs.h>
