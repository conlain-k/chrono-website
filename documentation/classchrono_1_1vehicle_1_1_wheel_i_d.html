chrono::vehicle::WheelID Class Reference
VEHICLE module

Description

Class to encode the ID of a vehicle wheel.

By convention, wheels are counted front to rear and left to right. In other words, for a vehicle with 2 axles, the order is: front-left, front-right, rear-left, rear-right.
