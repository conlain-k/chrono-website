chrono::irrlicht::ChIrrNodeAsset Class Reference
IRRLICHT module

Description

Class for adding Irrlicht visualization to a ChPhysicsItem.

This must be added as an 'asset' to the item (ex., a ChBody).

#include <ChIrrNodeAsset.h>
