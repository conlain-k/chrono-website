chrono::ChAssetLevel Class Reference

Description

Base class for grouping assets in a level.

The level is like a 'subdirectory'. A level can contain assets; amnog these, also further levels, etc. (but please avoid circular loops!) A level can have custom rotation and translation respect its parent level.
