chrono::ChLoaderXYZnode Class Reference

Description

FORCE AT XYZ NODE Loader for a constant force applied at a XYZ node.

Note that an equivalent shortcut is to set directly the node's own force member data as mynode->SetForce(), but this other approach is more object-oriented (es. you can apply multiple forces at a single node, etc)

#include <ChLoadsXYZnode.h>
